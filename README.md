## Task 3: SQL for Data Analysis

### Tools Used
- Oracle SQL (SQL*Plus)

### Description
This task demonstrates SQL queries for data analysis using an ecommerce-style dataset.

### Concepts Covered
- SELECT, WHERE, ORDER BY
- GROUP BY, HAVING
- Aggregate functions (SUM, AVG)
- Joins (INNER, LEFT)
- Subqueries
- Views
- Indexes for optimization
- NULL handling

### Outcome
Learned how to extract, analyze, and optimize structured data using SQL.
